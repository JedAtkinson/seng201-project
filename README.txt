Before doing any of the following steps ensure that you have extracted all the files

To open soruce code in eclips
    1. Click file, import, General, Project from folder or archive
    2. Select "SENG201 Project" file as import soruce
    3. Selct finish and the project should be imported
To run from imported eclips project
    1. open SENG201 project/scr/gui/GUIManager
    2. run file
To run game from jar file
    1. open cmd and cd to this directory
    2. run the command "java -jar abu84_jat200_MonsterFighter.jar"

Note javaDoc located in SENG201 project file along with source code
